 3. Implementasikan mekanisme penanganan kesalahan (error handling) pada setiap endpoint yang anda buat.
 4. Buat struktur folder ataupun code sebaik mungkin.- Soal Test:

    3. Buat system desgn untuk point 2.
    4. Buat endpoint yang mengelola proses integrasi dengan API eksternal.
